# Twillio Burner

A Twillio-based text forwarding application

## Requirements

- [nvm](https://github.com/creationix/nvm) or similar
- [foreman](https://github.com/ddollar/foreman) or similar

## Quick Start

    $ nvm use
    $ npm install
    $ cp .env.example .env

    $ foreman start -f Procfile.dev
